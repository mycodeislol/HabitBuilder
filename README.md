#HabitBuilder

Habit Builder is a full-stack web application designed to help users develop and maintain consistent daily habits. The platform allows users to create, track, and visualize their habit progress through an intuitive dashboard, supported by insightful data analytics and a clean, modern interface.

#Overview:
The project integrates a React-based frontend with a Spring Boot backend, creating a seamless experience between the client interface and the server-side logic. The application emphasizes user engagement, data-driven progress tracking, and modular design for scalability.

#Features:

User authentication and secure session handling

Dashboard for viewing and managing habits

Real-time progress visualization using interactive charts

Dynamic toast notifications and animations for user feedback

Modular and reusable frontend components

RESTful API integration between frontend and backend

Responsive and minimalistic design using Tailwind CSS

#Technology Stack
Frontend

React.js (Vite) – for high-performance UI development

Tailwind CSS – for responsive and utility-first styling

Recharts – for habit tracking data visualization

Axios – for communication with backend APIs

React Toastify – for notifications and alerts

Backend

Spring Boot (Java) – for REST API development

Spring Data JPA – for database interactions

Hibernate ORM – for entity management

MySQL / PostgreSQL – for persistent data storage

Spring Security – for authentication and access contro
